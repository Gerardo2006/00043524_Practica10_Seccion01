¿Cual es la diferencia entre autenticacion y configuracion?
R/ La autenticacion verifica la identidad de un usuario, mientras que la configuracion define como se comporta una aplicacion. Son procesos distintos: uno protege acceso, el otro ajusta funcionamiento.

¿Para que se usan los tokens en JWT?
R/ Los tokens en JWT se usan para identificar y autenticar usuarios de forma segura entre el cliente y el servidor.
